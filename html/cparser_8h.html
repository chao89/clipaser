char *  filename 
)

Load a command/config file to the parser.

A command/config file is just a text file with CLI commands. (One command per line.) The only difference is that submode is automatically exited if the indentation changes. This behavior is the same as Cisco CLI.

Parameters
parserPointer to the parser structure.
filenamePointer to the filename.
Returns
CPARSER_OK if succeeded; CPARSER_ERR_INVALID_PARAMS if the input parameters are NULL; CPARSER_NOT_OK if the file cannot be opened.
