Detailed Description

Test program for verifying parser FSM.

There are three states - WHITESPACE, TOKEN, ERROR and three types of inputs CHARACTER, ERASE and SPACE. The following is a complete list of state transition.

  1. (WHITESPACE, SPACE) -> WHITESPACE
  2. (WHITESPACE, ERASE) -> WHITESPACE
  3. (WHITESPACE, ERASE) -> TOKEN
  4. (WHITESPACE, CHAR) -> CHAR
  5. (WHITESPACE, CHAR) -> ERROR
  6. (TOKEN, SPACE) -> WHITESPACE
  7. (TOKEN, SPACE) -> ERROR
  8. (TOKEN, CHAR) -> TOKEN
  9. (TOKEN, CHAR) -> ERROR

(TOKEN, ERASE) -> TOKEN

  1. (TOKEN, ERASE) -> WHITESPACE
  2. (ERROR, SPACE) -> ERROR
  3. (ERROR, CHAR) -> ERROR
  4. (ERROR, ERASE) -> ERROR
  5. (ERROR, ERASE) -> WHITESPACE
  6. (ERROR, ERASE) -> TOKEN
