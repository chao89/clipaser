Complete one level in the parse tree.

There are serveral cases we will complete one level:

static void cparser_help_print_node ( cparser_t *  parser,
cparser_node_t *  node,
const int  add_lf,
const int  print_desc 
)
static

Print the description of a node out.

For keyword nodes, the keyword itself is printed. For parameter nodes, a string of the form <[type]:[parameter]> is printed. For end nodes, <LF> is printed. This call should never be invoked with a root node.

Parameters
parserPointer to the parser structure.
nodePointer to the node to be printed.
add_lf1 to add LF before printing the node; 0 otherwise.
print_desc1 to print the description of the node; 0 otherwise.
