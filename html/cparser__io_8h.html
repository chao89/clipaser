Detailed Description

Platform-dependent parser I/O routines header and definition.

Version
$Id: cparser_io.h 142 2011-09-01 05:31:27Z henry $

CLI parser library requires some platform-dependent API to provide access to the terminal. These API can vary greatly among different OSs.
